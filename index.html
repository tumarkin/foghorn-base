foghorn-base

Foghorn.Base defines the core kind-level economic types (EconType), dataset indices (EconIndices), and associated constraints used across the Foghorn ecosystem. It provides the type-safe foundations for the Foghorn DSL, which supports declarative, replicable econometric research.

Most end users will not interact with this library directly, but will instead import domain-specific Foghorn modules that build on top of these base types. Foghorn.Base is intended for library authors, contributors, and those extending the DSL.
